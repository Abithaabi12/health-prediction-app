# Health Prediction App

This is a simple health prediction web application built using Flask and SQLite.

## Features
- Add, update, delete patient records
- Predict health status based on input values
- Simple and clean UI using Bootstrap
- External API integration

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, Bootstrap

## How to Run
1. Install Flask
2. Run: python app.py
3. Open browser: http://127.0.0.1:5000
